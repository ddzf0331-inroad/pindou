# 拼豆设计使用手册

## 1. 打开软件

正式安装版优先从系统应用入口打开：

- macOS：把 `拼豆设计` 拖入“应用程序”后双击打开。
- Windows：安装后从桌面快捷方式或开始菜单打开。

安装版会把数据保存在系统用户数据目录，升级软件时不会覆盖你的色块库、图库和拼装进度。

如果使用开发/备用版本，在 macOS 上双击 `拼豆设计.app`。

如果打不开，也可以双击 `打开拼豆设计.command`。软件会自动尝试启动本机服务；如果服务启动不了，会自动打开静态版。

Windows 用户双击 `打开拼豆设计.bat`。

Linux 用户运行 `scripts/launch-unix.sh`。

打开后看左侧状态条：显示“Node 本机服务”或“Python 后备服务”时，开发/备用版本的数据保存在软件目录的 `data` 文件夹；安装版的数据保存在系统用户数据目录；显示“静态版”时，数据保存在当前浏览器里。

## 2. 第一次试用

1. 打开后先看“首页”，确认图库、拼装进度和低库存提醒。
2. 进入“工作台”。
3. 点击“使用示例图”。
4. 点击“生成像素画”。
5. 在右侧查看色块统计。
6. 点击“采纳当前结果”保存到图库。
7. 点击“导出生产文件”得到编号图、用量清单、坐标矩阵和分区打印版。

## 3. 导入自己的图片

在工作台点击“导入真实照片或卡通图”。

- 如果是照片，选择“真实照片”。
- 如果已经是卡通图，选择“卡通图”。

照片模式可以在 AI 设置中启用云端卡通化；如果没有配置 AI，软件会自动使用本地预处理。

如果要还原别人导出的图纸，点击工作台里的“解析图纸”。软件会只读取图纸上半部分的像素网格，按每个格子里的文字色号逐格 OCR 识别，解析结果会先进入当前工作台画布。确认无误后，再点击“采纳当前结果”保存到图库。

图纸解析需要服务器安装 Tesseract OCR。没有 OCR 时，软件会直接提示缺少 OCR 引擎，不会再用底色近似匹配生成错误结果。

## 4. 设置成品尺寸

在工作台设置像素画宽度和高度，也可以选择常用规格。

建议先用 `64 x 64` 或 `80 x 80` 试效果，再做更大的生产图。

如果希望控制生成时使用的具体色号，可以点击“生成色号清单”。软件可以按当前图片、裁剪和尺寸分析推荐色号，也可以手动勾选色号。清单里有勾选色号时，生成像素图会严格按这份清单匹配颜色。

## 5. 调整取图范围

“主体裁剪”有三种方式：

- 主体居中：适合大多数头像、玩具主体图。
- 完整画面：保留整张图。
- 手动裁剪：可以直接在预览图里拖动取景框，滚轮或触摸板缩放取景。

## 6. 管理实体色块

进入“色块库”。

可以设置：

- 可用色块上限
- 色块编号
- 色块名称
- 当前库存
- 补充入库数量
- 单个色块入库或扣减
- 批量设定、批量补充、批量扣减库存
- 色盘颜色
- 色彩地图选点
- RGB 数值
- 启用、禁用、删除

如果要批量导入，先点击“下载导入模板”，按模板里的编号、名称、RGB 格式填写后，再点击“导入色块库”选择 JSON 文件。

库存低于 100 个时，首页看板和色块库会显示补货提醒。拼装页里标记某个格子已经拼好后，对应色块库存会自动减少 1；如果取消标记，库存会加回 1。

如果要快速整理库存，可以在色块库点击“批量库存”，选择“批量设为”“批量补充”或“批量扣减”，并选择作用范围：全部可用色块、低库存色块或全部未删除色块。

删除是软删除，历史作品仍会保留当时的色块快照。

## 7. 生成和重新生成

点击“生成像素画”后，结果会进入候选状态。

如果不满意，点击“重新生成”。右侧会保留最近的候选历史，可以点回旧候选查看。

满意后点击“采纳当前结果”，作品会进入个人图库。

## 8. 手工编辑像素画

右侧工具包括：

- 高亮：查看同名同色的所有格子。
- 画笔：逐格改色。
- 填充：填充连通区域。
- 取色：从画布上取当前色块。

修改后需要再次点击“采纳当前结果”，图库里的作品才会更新。

工作台左侧菜单和“上传与配置”面板都可以通过箭头按钮收起，适合在编辑大图时给预览区留出更多空间。刷新页面后会保留上次的收起状态。

工作台的上传配置栏、右侧色块统计栏，以及色块库里的编辑区和色块列表都会独立滚动。色块很多时，可以只滚动侧栏，中间预览区不会跟着跑掉。

在 Mac 触摸板上，可以在预览区双指捏合放大或缩小，双指滑动移动画布。也可以按住空格拖动画布，或用鼠标中键拖动画布。

## 9. 查看统计和风险

右侧会显示本次用了哪些色块、每种用了多少块。

如果某些颜色和色块库差距较大，软件会显示色差风险，并给出建议色。可以点击“加入色库”，再重新生成。

## 10. 导出生产文件

点击“导出生产文件”会导出：

- 编号 PNG：整张编号图。
- 色块用量 CSV：每种色块需要多少块。
- 坐标矩阵 CSV：每个坐标对应哪个编号。
- 分区打印版 HTML：自动按 `32 x 32` 分区，适合打印或另存为 PDF。

## 11. 图库和备份

进入“图库”可以打开、导出、删除作品，也可以点击“开始拼装”或“继续拼装”进入拼装页。

## 12. 拼装页

拼装页用于跟进现实中的拼装进度，不会修改原始像素画。

- 点击像素格：标记或取消“已拼好”。
- 右侧统计：显示每种色块的总数、已拼和剩余数量。
- 点击色块：高亮图中所有同名同色的格子。
- 选中色块后拖拽框选：只把框选范围内同色的格子标记为已拼好，不影响其他颜色。
- 定位网格：可选择每 10 格或每 5 格显示粗分割线，也可以隐藏。
- 点击“定位”：从左上到右下寻找该色块下一个未拼好的格子，自动放大到能显示编号，并把位置居中。
- 点击“保存拼装进度”：手动保存当前已拼标记，下次从图库点击“继续拼装”可接着做。

拼装进度会自动跟随作品保存，也可以手动点击保存。刷新页面、重启软件、导出导入作品或完整备份恢复后，已标记的位置都会保留。

## 13. 图库和备份

建议定期点击“备份图库”，保存作品 JSON 文件。

如果换电脑，可以在新电脑里点击“导入作品”恢复。

如果需要连同色块库、色块上限、AI 设置和拼装进度一起保存，进入“AI 设置”，点击“备份全部数据”。完整备份不会明文导出 API Key，只会记录是否已经设置。

需要恢复整套数据时，进入“AI 设置”，点击“恢复完整备份”，选择之前导出的完整备份文件。恢复会替换当前色块库、图库和 AI 设置。

如果以前直接打开过静态版 `index.html` 并保存过色块或作品，改用本机服务版后，可以在“AI 设置”里点击“迁移静态版数据”，把浏览器本地存储里的旧数据搬到 `data` 文件夹。

## 14. AI 设置

进入“AI 设置”。

可以维护多个模型配置，并选择当前启用配置。

OpenAI 模式需要填写 API Key。自定义模型端点需要返回 `imageDataUrl` 或 `b64_json`。

如果没有 AI Key，也可以使用本地模拟模式。

## 15. 运行诊断

在 AI 设置页点击“运行诊断”，可以查看：

- 当前是静态版、Node 本机服务，还是 Python 后备服务。
- 数据保存目录。
- 色块、图库、设置文件是否已经保存。
- 当前启用的模型配置。
- API Key 是否已经保存。

遇到问题时，先运行诊断。

如果需要查看本机保存文件，点击“打开数据目录”。静态版没有数据目录，数据保存在浏览器本地存储中。
